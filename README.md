# 📱 Contact Management System

A simple Flask-based web app to manage contacts with CRUD operations.

---

## 🚀 Features

* ➕ Add Contact
* 📋 View Contacts
* ✏️ Edit Contact
* ❌ Delete Contact
* 🔍 Search Contacts
* 🎨 Styled with CSS

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML
* CSS

---

## 📂 Project Structure

```bash
contact_management_system/
│
├── app.py
├── templates/
│   ├── index.html
│   ├── add.html
│   └── edit.html
│
└── static/
    └── style.css
```

---

## ⚙️ Run Locally

```bash
pip install flask
python app.py
```

Open 👉 http://127.0.0.1:5000/
